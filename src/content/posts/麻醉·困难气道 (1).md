# 什么是困难气道？
指经过专业训练的有5年以上临床麻醉经验的麻醉科医师发生**面罩通气**困难或**插管**困难，或二者**兼具**的临床情况。 

# 面罩通气困难
1. 面罩通气困难原因：由于面罩封闭不严、大量气体泄露、通气存在巨大阻力，而使操作者无法给患者提供足够换气。  
2. 面罩通气困难预测指征：

# 困难气管插管
1. 指无论存在或不存在气道病理改变，有经验的麻醉科医师气管插管均需要三次以上努力。
2. 困难气管插管预测指征：

| **指征** | **提示困难插管的因素** |
| :---: | :---: |
| 既往史 | 困难插管史 |
| 上切牙的长度 | 相对较长 |
| 上下切牙间距 | 小于2指或小于3cm |
| 覆合牙 | 上切牙覆盖下切牙 |
| 颞下关节活动度 | 下切牙不能向前伸展到上切牙前方 |
| 下颌空间 | 小、坚硬、肿物侵入 |
| 颈椎活动范围 | 颏部不能触到胸部或不能伸展颈部 |
| 甲颏距离 | 小于3指（小于6cm） |
| Mallampati/Samsoon分级 | Mallampati/Samsoon分级Ⅳ级，舌体相对较大，<br/>不能看到悬雍垂 |
| 颈部 | 短、粗 |
| 腭部外形 | 高尖形、很狭窄 |
|||

# 如何处理已预测到的面罩通气困难或困难插管？
1. 最安全的处理方法：让患者清醒并保留自主呼吸。
2. 清醒插管的气道工具：支气管软镜和视频喉镜。
3. 如何选择：
+ a. 取决于患者因素，操作技能和设备可获得性。
+ b. 张口受限、舌体较大或颈部屈曲畸形固定的患者，支气管软镜可能更合适。
+ c. 气道出血患者，视频软镜可能更适合。
+ d. 已选用的清醒插管技术失败，应考虑替代方法。
+ e. 复杂的临床情况下，可以考虑同时使用视频喉镜和支气管软镜的清醒插管组合方法。
4. 气道表面麻醉效果良好的患者可以通过声门上气道进行纤支镜清醒插管，可能更利于维持气道通畅。
5. 清醒插管流程主要包括：镇静、表面麻醉，纤支镜/视频喉镜插管以及确认导管位置。

# 如何确认气管插管成功？
1. 看到气管导管通过声门进入到气管；
2. 将纤支镜插入到气管导管，从而确认气管软骨环及看到气管隆突；
3. 监测呼气末二氧化碳浓度，可以是定性或者定量，通常可以通过监护仪看到二氧化碳波形及数值。

# 困难插管的患者术后如何拔管？
拔管时需要评估气道的水肿程度，气管插管和手术操作都会造成舌根和喉部的肿胀，气道水肿在呼吸道阻塞时达到高峰。  
1. 有水肿风险的患者最好延长气管导管放置时间或采用气管造口术。一旦水肿消除，就可以考虑拔除气管导管或气管套管。
2. 气囊漏气实验可以用来评估气道是否存在水肿，此试验是在拔管前将气管导管套囊放气，可以观察到从气管导管周围溢出气体。临床上若无气囊漏气试验则不能保证安全拔管。
