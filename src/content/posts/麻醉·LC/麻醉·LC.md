# LC围手术期麻醉管理要点
1. 注意患者的液体管理；
2. LC手术时间短，维持用药尽量选择短效麻醉药物，如丙泊粉、瑞芬太尼泵注；
3. 手术医师在刺激胆囊时注意胆心反射；
4. 围术期患者的血压维持在入手术室稳定后血压上下浮动不超过25%，避免低血压和血压偏高；
5. 注意围手术期的气道管理，人工气腹期间应维持足够的通气量，维持呼气末CO<sub>2</sub>分压在35mmHg左右。

# LC患者在什么情况下会出现胆心反射？
只有在浅麻醉状态下会出现；麻醉状态下人体交感和副交感系统都处于相对抑制状态，一般不会出现明显的胆心反射。

# 迷走神经反射的防治
1. 加强术前检查和准备，麻醉前应用足量抗胆碱药；
2. 术中出现心动过缓应及早静注阿托品，伴有血压下降时加用麻黄碱；
3. 必要时应暂停手术刺激；
4. 还可采取预防措施，如用利多卡因局部表面麻醉或行腹腔神经丛阻滞。

# 腹腔镜手术需要深度肌肉松弛吗？
腹腔镜手术并不需要很深的肌肉松弛，但是良好的肌肉松弛可以在不增加腹内压的前提下提供更大的手术空间。

# 短小手术可以使用喉罩代替气管插管吗？
短小手术是可以选用喉罩代替气管导管行辅助通气的。在腹腔镜手术中应用喉罩有利有弊：a. 有利的是喉罩易于插入、对于喉部刺激小、术后咳嗽等并发症发生率低；b. 不利的是由于使用人工气腹，腹内压增高，气道压超过20cmH<sub>2</sub>O时喉罩可发生漏气，难以进行控制呼吸，不能保护气道避免反流、误吸。故腹腔镜手术中喉罩通气的应用推荐仅限于保留自主呼吸和较瘦的健康患者短时间控制呼吸。
